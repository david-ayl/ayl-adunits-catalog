An Adyoulike Adunit is a single type of ad that lets you describe the placement type, define it location and leverage inventory quality.

Here are 3 types of these adunits:

InFeed
InAd
InText
@@include("app/views/modules/components/section/header/header.html", { "adunit": "if", "title": "InFeed", "subtitle" : "The true premium native placement." } )

Native advertising was originally designed for this type of integration. InFeed adunit is tailored to reproduce the design of the website editorial content tiles.

Usual locations:

  • Home-pages, categories.
  • Editorial zones.

Keypoints:

  • Premium emphasis.
  • Requires a dedicated tailored template.
  • Limited audience - from 10 to 20% of site global.
@@include("app/views/modules/components/section/header/header.html", { "adunit": "ia", "title": "InAd", "subtitle" : "Global inventory coverage." } )

The inad adunits gather IAB compliant sizes.

Usual locations:

  • Run of site.
  • Advertising areas like sidebars, billboards, skyscrappers etc.

Keypoints:

  • Compliants sizes: not below 300x250 pixels
  • Globally covers site pages and maximizes site audience coverage.
  • Mostly used when it comes to header bidding.
@@include("app/views/modules/components/section/header/header.html", { "adunit": "it", "title": "InText", "subtitle" : "Dedicated to articles." } )

Provides a better site audience coverage. Article pages attract a significant audience, share - generally min 80% of no. unique visitors / impression. A Good integration with a good size.

Usuals locations:

  • Articles pages.
  • Between two paragraphs.

Keypoints:

  • Provides a better site audience coverage - article pages attract a significant audience.
  • A good integration with a good size.
