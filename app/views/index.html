An Adyoulike Adunit is a single type of ad that lets you describe the placement type, define it location and leverage inventory quality.
Here are 3 types of these adunits : InFeed, InAd and InText.

@@include("app/views/modules/components/section/header/header.html", { "adunit": "if", "title": "InFeed" } )

True premium native placement, native advertising was originally designed for this type of integration. InFeed adunit is tailored to reproduce the design of the website editorial content tiles.

  • Premium emphasis from the homepage or site category, really appreciated by advertisers.
  • Can be used in addition to the InText format to cover 100% of the site audience.
@@include("app/views/modules/components/section/header/header.html", { "adunit": "ia", "title": "InAd" } )

  • Only accepts placements of size 300x250px or 300x600px - MPU type.
  • Globally covers site pages: homepage / category / article and maximizes site audience coverage.
  • Mostly used when it comes to header bidding.
@@include("app/views/modules/components/section/header/header.html", { "adunit": "it", "title": "InText" } )

Located between two paragraphs of the article.

  • Provides a better site audience coverage - article pages attract a significant audience.
  • A good integration with a good size.
